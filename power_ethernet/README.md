**Beschreibung:**  
Der power_ethernet Check dient zur Überwachung von PowerOverEthernet in Cisco und HP Switchen.

**Installation:**  
Manuell:  
CMK-Server: den Check aus "checks/power_ethernet" nach "/omd/sites/SITE/local/share/check_mk/checks/" kopieren
