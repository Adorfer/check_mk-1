**Beschreibung:**  
Der fc_ports Check dient zur Überwachung von FiberChannel Ports.
Im Jahr 2014 gab es kein FC-Check für Brocade/IBM der die 64-Bit Counter abfragen konnte, daher wurder der fc_ports Check angepasst

**Installation:**  
Manuell:  
CMK-Server: den Check aus "checks/fc_ports " nach "/omd/sites/SITE/local/share/check_mk/checks/" kopieren
