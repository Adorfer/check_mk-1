**Beschreibung:**  
Der cisco_stack Check dient zur Überwachung von Cisco Stack Switchen (mehrer einzelne Switche werden zu einem logischen Switch verbunden)

**Installation:**  
Manuell:  
CMK-Server: den Check aus "checks/cisco_stack" nach "/omd/sites/SITE/local/share/check_mk/checks/" kopieren
