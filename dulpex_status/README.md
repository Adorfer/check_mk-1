**Beschreibung:**  
Der dulpex_status Check dient zur Überwachung des Dulpex-Port-Status auf Cisco Switchen.
Wenn sich ein Port im HalfDuplex-Modus befindet geht der check auf WARN

**Installation:**  
Manuell:  
CMK-Server: den Check aus "checks/dulpex_status" nach "/omd/sites/SITE/local/share/check_mk/checks/" kopieren
